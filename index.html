Amigo

Amigo is a user-interface engine, which is supposed to be easy to set up. I am making it to learn about programming using C++ and OpenGL.

How it works

Once you have your MenuSystem created, you can use MenuSystem->AddMenu to add a menu. You can add as many menus as you want. You can then use Menu->AddItem to add items. Items can be modified at any time as needed, using functions like ->SetPosition, ->SetSize, ->SetText, and so on.

The entire UI uses only one small texture.

I'm using Visual Studio 2015 to make it.
